# Expense Tracker

**Автор:** [Михайлов Иван]

## Описание
Простое GUI-приложение на Python для учета личных расходов. Позволяет добавлять записи, фильтровать их по датам и автоматически суммировать затраты. Данные сохраняются в JSON-файл.

## Как использовать
1. Установите Python 3.x.
2. Запустите программу: `python expense_tracker.py`.
3. Введите сумму, выберите категорию и дату. Нажмите "Добавить".
4. Для фильтрации введите диапазон дат в поля "С" и "По" и нажмите "Применить".

## Тестирование
- Попробуйте ввести отрицательную сумму — программа выдаст ошибку.
- Попробуйте ввести дату в формате ДД.ММ.ГГГГ — программа потребует ГГГГ-ММ-ДД.
